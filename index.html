Roots CSH File

Reflections:

The goal of this assignment was to gain experience writing a test automation for the roots.crun program.
The specifications were established by running the code supplied to me and taking note on how the program is working.Then the testcases were created after getting a good understanding of the program leads to better quaility test cases.Repeatability is established in the test automation itself. The test automation allows for multiple programs to be tested with the test cases that have been created in this assignment. The test automation creates a utr file wich allows for accountability. The utr is a test report file which gives the results of the testing performed on the program.Having everything in one place, a place to run the test cases, and a way to look at the results of the test.
